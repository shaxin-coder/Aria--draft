# Delete a File API – Test Cases

| Spec           | `File_Delete_API_Interface.md` |
|----------------|--------------------------------|
| Endpoint       | DELETE `/temp-applications/{tmp_app_id}/assets/{document_id}` |
| Authentication | Bearer token                    |
| Note           | DB update: documents table DELETE. GCS object deletion. |

---

## API overview

**API name:** Delete a File API  
**Description:** Permanently removes a file (both metadata and the actual object in Google Cloud Storage).

---

## Sample success response

**HTTP 204 No Content** – No response body is returned for successful deletion.
